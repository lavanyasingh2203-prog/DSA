# 100_Days_of_DSA
Welcome to my DSA practice repository! This is a collection of solutions and exercises on various data structures and algorithms topics. The goal is to improve problem-solving skills and deepen my understanding of DSA concepts.

Topics Covered
Arrays and Strings
Linked Lists
Stacks and Queues
Trees (Binary Trees, BSTs)
Graphs (BFS, DFS)
Sorting Algorithms
Searching Algorithms

#Getting Started
To run the code examples, ensure you have Python (or the language of your choice) installed. Each solution is organized by topic, and I’ve added comments for better understanding.

How to Contribute
